# Virus
LAN multiplayer game where two or more players have to fight waves of enemies trying to survive as long as possible.

# How To Play

<h2>Movement</h2>
<ul>
  <li>W: Up</li>
  <li>A: Left</li>
  <li>S: Down</li>
  <li>D: Right</li>
</ul>

<h2>Attack</h2>
<ul>
  <li>Mouse pointer: Aim</li>
  <li>Left mouse button: Fire</li>
  <li>E: Switch weapon</li>
</ul>
